# React + Tailwind Playground

A lightweight, zero-config environment for rapid prototyping using React and Tailwind CSS directly in the browser via CDNs.

## Features

- **React 18:** Functional components and hooks (`useState`) used via UMD builds.
- **Tailwind CSS:** Modern utility-first styling with the Tailwind Play CDN.
- **Babel Standalone:** JSX transpilation in the browser, no build step required.
- **Responsive Design:** Polished, mobile-friendly UI out of the box.

## Getting Started

Simply open the `index.html` file in any modern web browser.

```bash
# Example: opening with Live Server or similar
npx serve .
```

## Project Structure

- `index.html`: The main entry point containing the HTML structure, Tailwind configuration, and React application logic.

## Usage

The current application features a simple interactive counter to demonstrate state management and Tailwind transitions. You can modify the `<script type="text/babel">` section in `index.html` to start building your own components.
